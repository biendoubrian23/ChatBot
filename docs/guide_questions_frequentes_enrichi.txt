SECTION 1 : FORMATS DE LIVRE - CHOIX DU FORMAT
======================================================================

QUEL FORMAT CHOISIR POUR IMPRIMER MON ROMAN ?
---------------------------------------------

Pour imprimer un roman, voici les recommandations de CoolLibri :

LE FORMAT 16×24 cm EST LE FORMAT LE PLUS ADAPTÉ POUR UN ROMAN.
Le format 16×24 cm (16 x 24 cm) est le format recommandé par excellence pour les romans car :
- Il offre un excellent confort de lecture
- C'est le format le plus utilisé par les éditeurs professionnels
- Il permet une bonne mise en page du texte
- Le rendu est professionnel et élégant

Le format 11×17 cm (poche) est aussi une option :
- Le format 11×17 cm (format poche) est plus compacte
- C'est une option plus économique pour les romans
- Idéal pour les livres de poche personnels

Le format A5 portrait (14,8×21 cm) convient également aux romans et guides.

IMPORTANT : Pour un roman, évitez les formats paysage ou carrés qui ne sont pas adaptés à la lecture de texte continu.


QUEL EST LE MEILLEUR FORMAT POUR UN LIVRE PHOTO ?
-------------------------------------------------

Pour un livre photo, voici les formats recommandés par CoolLibri :

LE FORMAT 21×21 cm (CARRÉ) EST LE FORMAT STAR POUR LES LIVRES PHOTOS.
Le format carré 21×21 cm est souvent utilisé pour un livre photo car :
- Il permet de mettre en valeur les images de manière équilibrée
- C'est le format idéal pour les albums photos
- Parfait pour les livres de mariage
- Excellent pour les portfolios artistiques

Le format A4 portrait (21×29,7 cm) est aussi recommandé pour les beaux livres et albums.

Le format A4 paysage (29,7×21 cm) est recommandé pour :
- Les beaux livres photos avec images panoramiques
- Les albums avec photos de paysages
- Les portfolios grand format

PAPIER RECOMMANDÉ POUR LES LIVRES PHOTOS :
Utilisez du papier COUCHÉ SATIN 115g blanc pour un rendu optimal des photos couleur.
Le papier couché satin 115g est idéal pour mettre en valeur vos photos en couleur.

ATTENTION : Les formats A5 Paysage peuvent être utilisés pour des livres photos mais le format 21×21 cm (carré) est préférable car il offre un meilleur équilibre visuel.

SECTION 2 : TYPES DE PAPIER - CHOIX DU PAPIER INTÉRIEUR
======================================================================

QUELS TYPES DE PAPIER PROPOSEZ-VOUS POUR L'INTÉRIEUR DU LIVRE ?
---------------------------------------------------------------

CoolLibri propose 4 TYPES DE PAPIER pour l'intérieur du livre.
Voici la liste complète des 4 papiers disponibles avec leurs caractéristiques :

1. PAPIER STANDARD 80g BLANC
   - Caractéristique : Équivalent au papier d'imprimante A4
   - C'est le papier le plus économique
   - Texture : Standard, classique
   - Usage recommandé : Romans et livres de poche en noir et blanc
   - Avantage : Prix attractif

2. PAPIER BOUFFANT 90g BLANC (doux, cotonneux)
   - Caractéristique : Légèrement cotonneux avec texture douce au toucher
   - Sensation au toucher : Douceur, sensation cotonneuse
   - Usage recommandé : Romans et livres de poche
   - ATTENTION : Il n'est PAS adapté lorsqu'il y a des photos en couleur
   - Le rendu des couleurs n'est pas optimal sur ce papier

3. PAPIER BOUFFANT 90g CRÈME (rendu ancien)
   - Caractéristique : Légèrement cotonneux avec couleur crème
   - Effet visuel : Rendu ancien, vintage, traditionnel
   - Sensation au toucher : Douceur, sensation cotonneuse
   - Usage recommandé : Romans avec aspect traditionnel, livres classiques
   - ATTENTION : Il n'est PAS adapté lorsqu'il y a des photos en couleur

4. PAPIER COUCHÉ SATIN 115g BLANC (lisse, idéal photos couleur)
   - Caractéristique : Toucher lisse et satiné
   - C'est LE PAPIER IDÉAL POUR LES LIVRES PHOTOS
   - C'est LE PAPIER RECOMMANDÉ pour les ouvrages avec images couleur
   - Usage recommandé : Livres photos, bandes dessinées, albums illustrés
   - Avantage : Met parfaitement en valeur vos photos en couleur
   - Rendu optimal des couleurs grâce à sa surface lisse

RÉCAPITULATIF DES 4 PAPIERS COOLLIBRI :
- Standard 80g blanc = économique, pour texte noir et blanc
- Bouffant 90g blanc = doux cotonneux, pour texte (pas photos couleur)
- Bouffant 90g crème = doux cotonneux rendu ancien, pour texte (pas photos couleur)
- Couché satin 115g blanc = lisse, IDÉAL pour photos couleur et livres illustrés

SECTION 3 : RENDU 3D ET LIVRE VIRTUEL
======================================================================

LE RENDU 3D SUR LE SITE NE RESSEMBLE PAS À CE QUE J'ATTENDS, EST-CE NORMAL ?
----------------------------------------------------------------------------

IMPORTANT - INFORMATION ESSENTIELLE :
Le rendu 3D et le livre virtuel sont des APERÇUS NON CONTRACTUELS.

Voici ce qu'il faut savoir sur le rendu 3D CoolLibri :

1. LE RENDU 3D EST NON CONTRACTUEL
   - Le rendu 3D est un aperçu indicatif seulement
   - Il ne matérialise PAS les marges de fabrication exactes
   - Des différences peuvent exister entre l'aperçu et l'impression finale
   - Les couleurs à l'écran peuvent différer des couleurs imprimées
   - Certains détails peuvent apparaître différemment

2. CE QUE LE RENDU 3D PERMET DE VISUALISER :
   - L'aspect général du livre (couverture, dos, épaisseur)
   - L'enchaînement des pages
   - Le positionnement global des éléments

3. POUR AVOIR UNE IDÉE EXACTE DU RENDU FINAL :
   - CONSEIL : Imprimez une ou deux pages en taille réelle pour vérifier
   - Utilisez notre outil de mise en page en ligne pour personnaliser la maquette
   - Cela permet d'avoir un aperçu plus réaliste

4. POURQUOI DES DIFFÉRENCES ?
   - Le rendu écran dépend de votre moniteur
   - Les couleurs peuvent varier selon l'écran utilisé
   - L'impression physique utilise des encres CMJN différentes des couleurs RVB écran
   - Les marges de fabrication ne sont pas visibles sur l'aperçu

CONSEIL IMPORTANT : Pour une meilleure précision sur le résultat final de votre livre imprimé, nous vous recommandons d'imprimer une page test en taille réelle sur votre imprimante personnelle avant de valider la commande.

SECTION 4 : EMAILS AUTOMATIQUES APRÈS COMMANDE
======================================================================

POURQUOI JE REÇOIS DES EMAILS ALORS QUE J'AI DÉJÀ PASSÉ COMMANDE ?
------------------------------------------------------------------

EXPLICATION COMPLÈTE DU SYSTÈME D'EMAILS AUTOMATIQUES :

Le système CoolLibri envoie des emails automatiques si des projets sont encore enregistrés comme "EN COURS DE PRÉPARATION" dans votre espace utilisateur.

POURQUOI RECEVEZ-VOUS CES EMAILS ?
Cela arrive même si :
- Une commande a déjà été validée pour un autre projet
- Le projet concerné n'a finalement pas été utilisé
- Plusieurs versions du projet ont été créées avant la commande finale
- D'anciens projets non commandés sont encore dans votre espace

LES PROJETS "EN COURS DE PRÉPARATION" DÉCLENCHENT CES EMAILS.
C'est le système automatique qui détecte ces projets non finalisés et envoie des rappels.

QUE FAIRE ?
1. Vous pouvez IGNORER ces messages s'ils concernent des anciens projets
2. Vérifiez que votre commande est bien validée dans "Mon compte" > "Mes commandes"
3. Si un numéro de commande apparaît, la commande est bien enregistrée

SOLUTION POUR ÉVITER CES EMAILS À L'AVENIR :
SUPPRIMEZ VOS ANCIENS PROJETS NON UTILISÉS :
1. Ouvrir votre espace projet
2. Repérer les projets non commandés (en cours de préparation)
3. Les supprimer

Cela permet :
- D'éviter l'envoi automatique d'emails de rappel
- De rendre votre espace plus clair
- De réduire les risques de confusion entre différentes versions

COMMENT VÉRIFIER SI LA COMMANDE EST BIEN ENREGISTRÉE ?
1. Connectez-vous à votre compte CoolLibri
2. Cliquez en haut à droite sur "Mon compte"
3. Ouvrez la section "Mes commandes"
4. Si un numéro de commande apparaît, la commande est validée

SECTION 5 : ANNULATION ET DROIT DE RÉTRACTATION
======================================================================

PUIS-JE ANNULER MA COMMANDE APRÈS VALIDATION, J'AI 14 JOURS DE RÉTRACTATION ?
-----------------------------------------------------------------------------

INFORMATION LÉGALE TRÈS IMPORTANTE :

LE DROIT DE RÉTRACTATION DE 14 JOURS NE S'APPLIQUE PAS AUX LIVRES COOLLIBRI.

Pourquoi le droit de rétractation de 14 jours ne s'applique pas ?
- Les livres CoolLibri sont des PRODUITS PERSONNALISÉS
- Ils sont fabriqués selon VOS spécifications personnelles
- Conformément à l'article L221-28 du Code de la consommation
- Les produits personnalisés fabriqués sur mesure sont exclus du droit de rétractation

CONSÉQUENCE : Une fois la commande VALIDÉE, elle ne peut PAS être annulée.

CEPENDANT, vous disposez d'un DÉLAI DE RÉCLAMATION de 3 JOURS OUVRABLES APRÈS LIVRAISON.

Ce délai de réclamation permet de signaler :
- Un problème de qualité d'impression
- Un défaut sur le produit reçu
- Une erreur de fabrication

COMMENT FAIRE UNE RÉCLAMATION ?
- Délai maximum : 3 jours ouvrables après réception
- Email : contact@coollibri.com
- Documents à joindre : photos du problème + numéro de commande

PASSÉ LE DÉLAI DE 3 JOURS OUVRABLES, AUCUNE RÉCLAMATION N'EST ACCEPTÉE.

EN CAS D'ERREUR AVANT IMPRESSION :
Si vous avez validé une commande avec une erreur et que vous vous en rendez compte IMMÉDIATEMENT, contactez rapidement contact@coollibri.com. Plus la demande est envoyée tôt, plus les chances d'intervention avant impression sont élevées.

SECTION 6 : RELIURE REMBORDÉ - FORMATS DISPONIBLES
======================================================================

QUELS SONT LES FORMATS DISPONIBLES POUR LA RELIURE REMBORDÉ ?
-------------------------------------------------------------

IMPORTANT - FORMATS LIMITÉS POUR LA RELIURE REMBORDÉ :

Pour un livre avec une reliure en Rembordé (couverture cartonnée rigide type BD), les formats techniquement possibles sont UNIQUEMENT :

1. A4 portrait (21×29,7 cm)
2. A4 paysage (29,7×21 cm)  
3. 21×21 cm (format carré)

CE SONT LES 3 SEULS FORMATS DISPONIBLES POUR LA RELIURE REMBORDÉ.
Il n'y a que 3 formats : A4 portrait, A4 paysage et 21×21 cm.

ATTENTION : Contrairement aux autres reliures, la reliure Rembordé n'est pas disponible en format A5, ni en format 11×17, ni en format 16×24.

Pour les autres formats (poche, A5, 16×24...), vous devez choisir une autre reliure :
- Dos carré collé
- Spirale
- Agrafé

Nombre de pages pour le Rembordé :
- Minimum : 24 pages
- Maximum : 100 à 150 pages selon le papier choisi
