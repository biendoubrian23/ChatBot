QUESTION : L'impression de mon livre est floue et de mauvaise qualité, que puis-je faire ?
CATÉGORIE : Réclamations - Qualité d'impression
---

RÉPONSE ATTENDUE :
Si l'impression de votre livre est floue, vérifiez que votre fichier soit en haute 
résolution (300 dpi). S'il ne l'est pas, pensez à utiliser un fichier de meilleure 
qualité pour vos prochaines impressions. Si votre fichier est bien en 300 dpi, 
contactez-nous à contact@coollibri.com en précisant votre numéro de commande, une 
photo du problème et une brève description afin que nous puissions ouvrir un incident.

RÉSOLUTION DES PROBLÈMES DE QUALITÉ D'IMPRESSION :
1. Vérifier la résolution du fichier source (minimum 300 dpi requis)
2. Si fichier non conforme : utiliser un fichier de meilleure qualité
3. Si fichier conforme (300 dpi) : contacter le service client
4. Pour ouvrir un incident, fournir :
   - Numéro de commande
   - Photo du problème
   - Description brève du défaut

Nous sommes désolés, mais si les erreurs étaient déjà présentes dans votre fichier,

POLITIQUE SUR LES ERREURS DE FICHIER CLIENT :

- Si l'erreur provient du fichier client : AUCUN remboursement possible
- Le client est responsable du contenu de son fichier
- Assistance disponible pour corriger le fichier pour une future commande
- Contact : 05 31 61 60 42 ou contact@coollibri.com

---

RÉPONSE ATTENDUE :
Bonne question ! Le droit de rétractation standard de 14 jours (article L221-28 du Code 
de la consommation) NE s'applique PAS aux livres CoolLibri puisqu'ils sont des produits 
PERSONNALISÉS fabriqués sur mesure. 

CEPENDANT, voici ce qui peut se passer :

SI L'IMPRESSION N'A PAS COMMENCÉ :
   - Vous avez une CHANCE d'annulation
   - Mais il faut agir TRÈS rapidement
   - Contactez immédiatement le service client
   - Aucun délai légal de 14 jours : vous devez vous dépêcher
   - Remboursement possible si annulation réussie

SI L'IMPRESSION A DÉJÀ COMMENCÉ :
   - Annulation IMPOSSIBLE (c'est physiquement irréversible)
   - Solutions alternatives possibles (remboursement partiel, code promo)

COMMANDE D'ACTION URGENTE :
Contactez-nous AU PLUS VITE :
   - Téléphone (PRIORITAIRE) : 05 31 61 60 42
   - Email : contact@coollibri.com
   - Fournissez votre numéro de commande
   - Nous vérifierons si l'impression a commencé
   - Votre seule chance c'est la RAPIDITÉ

---

RÉPONSE ATTENDUE :

RÈGLES DE MODIFICATION DE FICHIER :
- Modification possible UNIQUEMENT avant paiement et validation finale
- Aller dans "Mes projets" pour modifier le fichier

QUESTION : Mon livre est déjà imprimé, puis-je annuler et être remboursé ?
CATÉGORIE : Annulation - Annulation après impression
---

RÉPONSE ATTENDUE :
Malheureusement, NON. Une fois l'impression commencée, il n'y a pas d'annulation possible. 
Voici pourquoi et ce qui peut être fait :

POURQUOI PAS D'ANNULATION APRÈS IMPRESSION ?
   - L'impression est un processus PHYSIQUE et IRRÉVERSIBLE
   - Les encres et les matériaux sont déjà appliqués au papier
   - Les coûts de production (encre, papier, électricité, main-d'œuvre) ont été engagés
   - Il n'est physiquement pas possible de « dé-imprimer » un livre
   - C'est la même raison pour laquelle les droits légaux de rétractation ne s'appliquent 
     pas aux produits personnalisés (loi française L221-28)

MAIS NOUS PROPOSONS DES SOLUTIONS :
   Si votre demande d'annulation est justifiée (erreur de notre part, mauvaise qualité 
   détectée, etc.), contactez notre service client avec :
   - Votre numéro de commande
   - Une explication claire de votre situation
   - Photos si problème de qualité
   
   Nous pouvons offrir :
   - Remboursement PARTIEL (après déduction des frais de production réellement engagés)
   - Code promo pour une future commande
   - Nouvelle commande corrigée gratuitement ou à prix réduit

COMMENT CONTACTER LE SERVICE CLIENT :
   - Téléphone : 05 31 61 60 42
   - Email : contact@coollibri.com
   - Mentionnez que votre impression a déjà commencé ou est terminée

SECTION 4 : PROBLÈMES TECHNIQUES
================================================================================

---
QUESTION : Le site refuse mon fichier, pourquoi ?
CATÉGORIE : Technique - Fichier refusé
---

RÉPONSE ATTENDUE :
Si votre fichier n'est pas accepté, c'est probablement parce qu'il n'est pas au 
format pris en charge (nous acceptons les fichiers Word ou PDF). Maintenant, si 
l'analyse du fichier indique des erreurs (en rouge), lisez les consignes qui vous 
aideront à les corriger. Si vous ne parvenez toujours pas à résoudre le problème, 
contactez-nous par email à contact@coollibri.com ou au 05 31 61 60 42.

CAUSES FRÉQUENTES DE REFUS DE FICHIER :
1. Format non supporté (acceptés : Word, PDF)
2. Résolution trop faible (minimum 300 dpi pour les images)
3. Format de page incorrect (doit correspondre au format commandé)
4. Marges insuffisantes (2cm minimum recommandé)
5. Erreurs détectées par l'analyseur (affichées en rouge)

SOLUTION :
- Lire attentivement les messages d'erreur en rouge
- Suivre les consignes de correction
- Si problème persistant : contacter contact@coollibri.com ou 05 31 61 60 42

---
QUESTION : Les couleurs de mon livre sont différentes de celles de mon écran, c'est normal ?
CATÉGORIE : Technique - Différence de couleurs
---

RÉPONSE ATTENDUE :
Les couleurs de votre fichier sur écran peuvent légèrement différer de celles sur 
le papier en raison des réglages et des caractéristiques de votre moniteur. Si vous 
constatez une différence importante, vous pouvez nous contacter au 05 31 61 60 42 
ou par email à contact@coollibri.com en envoyant des photos et votre numéro de 
commande afin d'étudier votre incident.

EXPLICATION TECHNIQUE DES DIFFÉRENCES DE COULEURS :
- Écran = affichage RVB (Rouge, Vert, Bleu)
- Impression = CMJN (Cyan, Magenta, Jaune, Noir)
- Les moniteurs non calibrés peuvent afficher des couleurs différentes
- Des variations de 7% sont considérées comme normales
- Le type de papier influence aussi le rendu des couleurs
- Conseil : imprimer une page test avant commande importante

---
QUESTION : Comment obtenir une facture pour ma commande ?
CATÉGORIE : Paiement - Demande de facture
---

RÉPONSE ATTENDUE :
La facture de votre commande est disponible directement dans votre espace client CoolLibri. 
Pour la télécharger :

PROCÉDURE COMPLÈTE POUR TÉLÉCHARGER UNE FACTURE :
1. Connectez-vous à votre compte sur le site CoolLibri
2. Cliquez sur "Mon Compte" en haut à droite de la page
3. Accédez à la section "Mes commandes"
4. Pour chaque commande, vous verrez un bouton ou lien "Télécharger la facture"
5. Cliquez dessus pour obtenir votre facture au format PDF

La facture est généralement disponible quelques heures après la validation de votre 
paiement. Elle contient tous les détails de votre commande (numéro, date, montants, 
TVA) nécessaires pour votre comptabilité.

NOTE IMPORTANTE : 
- Si la facture n'apparaît pas dans votre espace client après 24h, contactez 
  contact@coollibri.com avec votre numéro de commande
- Pour les professionnels avec compte PRO, une facture mensuelle regroupée peut 
  être disponible selon vos options
